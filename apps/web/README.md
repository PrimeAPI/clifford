# Web UI

Next.js-based admin interface.

## Features

- Create new runs
- View run details
- Monitor run steps in real-time
- Minimal, functional MVP UI

## Environment Variables

- `NEXT_PUBLIC_API_URL` - API server URL (default: http://localhost:3000)

## Development

```bash
pnpm dev
```

Visit http://localhost:3001 (or configured port)
